Casa automatica

Para este projecto usei a linguagem Java para criar a interface grafica que cria a interação entre o usuario e o hardware da casa. Para o Hardware da casa foi usado o Arduino Uno e alguns modulos como o Servo, RFID, Sensor Ultrasonico,etc. O objectivo principal deste projecto é demonstrar a ligação entre software e hardware e demonstrar tambem os principios basicos de uma casa inteligente(Smart home).

Esta é a fase final do projecto, podemos ver o funcionamento do mecanismo de abertura da porta principal e a detecção de obstaculos que é feita. Este foi um projecto de muito aprendizado e foi possivel aplicar varios conceitos da area de tecnologia. Infelizmente não foi possivel aplicar todas as funcionalidades que tinha em mente por falta de recursos, mas pretendo dar continuidade assim que for possivel.
